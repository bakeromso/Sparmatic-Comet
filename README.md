# Sparmatic-Comet
Open Source Firmware for cheap "Sparmatic Comet" thermostat

## Required Features
- [ ] Detect Valve Min/Max Position
- [ ] Set Valve Position (Percent) 
- [ ] Read Temperature
- [ ] Button Inputs
- [ ] Control Display
- [ ] Battery Voltage / Power loss detection
